1. Run `$ make`
2. Run `$ ./main > results.txt`


In `results.txt`:
* `0 -> 1`, `0` indicates walking, and `1` indicates speed 1
* `f1` indicates step up
* `f2` indicates step down
* Generally, any discontinuity represents a misclassifcation
